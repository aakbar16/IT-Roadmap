## Overview:

This roadmap provides a structured path to becoming a Technical Support Specialist, covering essential tools, systems, and troubleshooting skills.
It is organized into phases, starting with foundational knowledge and progressing to job-ready and advanced concepts.

Follow each step to build practical, industry-relevant skills needed for real technical support roles. This roadmap is divided into three phases, with an overall overview and a detailed weekly and daily path.


## Phase 1: Weekly Breakdown  

| Week     | Focus Area | Primary Cisco Courses | Detailed Tasks & Practical Labs
| -------- | ------- |  -------                 |  ------- |
| 1 | The Workstation & OS| Computer Hardware Basics, Operating Systems Basics |  Study internal components (RAM, CPU, Motherboard). Lab: Use a Virtual Machine to install Windows 11 and explore the Registry and Event Viewer. |
| 2 | Networking Fundamentals | Networking Basics, Network Addressing | Master the OSI Model and TCP/IP. Lab: Practice cabling standards and use ping, tracert, and ipconfig to map a home network. |
| 3 | Network Config & Troubleshooting | Network Support and Security, [BETA] Subnetting Mastery | Learn DHCP, DNS, and IP Addressing (IPv4/IPv6). Lab: Use Cisco Packet Tracer to build a basic local area network (LAN). |
| 4 | Linux & Command Line | Linux Unhatched, Linux Essentials | Learn file system navigation, permissions (chmod), and basic shell scripting in Bash. Lab: Install Ubuntu Linux and perform updates via terminal. |
| 5 | Security & User Access | Introduction to Cybersecurity, Endpoint Security | Understand firewalls, VPNs, and Malware protection. Lab: Practice setting up a basic software firewall and identifying phishing headers. |
| 6 | The Help Desk Workflow | IT Customer Support Basics, IT Troubleshooting Skills | Study the ticketing lifecycle (SLA, Escalation). Lab: Document a "mock ticket" from identification to resolution using a free tool like Trello or Jira. |

| Week     | Focus Area | Primary Cisco Courses | Detailed Tasks & Practical Labs

|3| Modern Mgmt (Intune) | Microsoft Intune Training  | Transition to cloud-based management for remote laptops and mobile devices. |
|4| Cloud & M365 | Microsoft 365 Certified: Fundamentals | Manage Exchange Online (Outlook) and Microsoft Teams settings. |

|6| Security Operations | Endpoint Security Fundamentals | Deploy Trellix or CrowdStrike agents and monitor for security threats. |
